# Slime Chunk Mapping - 一款由CreatorCSIE制作的小程序

**一种使用简单程序直观查看史莱姆区块及其坐标的解决方案**

# 功能
* 你可以单击上面的方块■得知坐标。
* 超过X/Z±12550820坐标不再绘制点。
* 如果你不知道坐标，可以使用模糊搜索来定位其最近的史莱姆区块。
* 随机种子功能。
* 你可以输入你自己的种子。
